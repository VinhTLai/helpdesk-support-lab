# Help Desk Support Simulation Lab

## Overview
This project simulates a real-world IT Help Desk environment where common user issues are logged, troubleshooted, and resolved using Windows administrative tools.

## Skills Demonstrated
- Help Desk ticketing and documentation
- Windows 11 troubleshooting
- User account management
- Networking fundamentals
- System performance analysis

## Tools Used
- Windows 11
- VirtualBox
- PowerShell
- Event Viewer
- Task Manager
- Device Manager
- osTicket
- XAMPP

## Lab Setup
- 1 Windows VM (Help Desk Technician)
- 1 Standard User account
- Local troubleshooting environment

## Tickets Resolved
- Password resets and account issues
- Network connectivity problems
- Printer and device errors
- Software installation and crashes
- Disk space and performance issues

## What I Learned
This project improved my troubleshooting methodology, documentation habits, and ability to communicate technical solutions clearly.
