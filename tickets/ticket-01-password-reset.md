# Ticket 01: Password Reset

## Issue
User was unable to log into their account after forgetting their password.

## Troubleshooting Steps
1. Logged in as ITAdmin.
2. Opened Local Users and Groups.
3. Reset the user's password.
4. Enabled "User must change password at next login."

## Tools Used
- Local Users and Groups
- Computer Management

## Resolution
Password was successfully reset and user was able to log in.

## What I Learned
This reinforced the importance of identity management and secure password practices.
