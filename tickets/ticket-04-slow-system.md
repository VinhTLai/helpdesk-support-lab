# Ticket 04: Slow System

## Issue
User reported that the computer was running slowly and applications were taking a long time to open.

## Environment
- Device: Windows 11
- User Account: TestUser1

## Troubleshooting Steps
1. Opened Task Manager to review CPU, memory, and disk usage.
<img width="1024" height="719" alt="high_system_usage" src="https://github.com/user-attachments/assets/acb5d341-34ee-468f-bbda-3cc8a888aa1e" />
   
3. Identified unnecessary startup applications.
4. Disabled non-essential startup programs.
<img width="1024" height="720" alt="disabled_startup_apps" src="https://github.com/user-attachments/assets/1e75d1e0-7994-4b21-90ae-a4e088a93086" />

6. Restarted the system to apply changes.

## Tools Used
- Task Manager

## Resolution
System performance improved after reducing startup applications and restarting the computer.
<img width="1024" height="721" alt="task_manager_after_fixes" src="https://github.com/user-attachments/assets/15c41630-7a98-452e-b71d-e6885d721af0" />

## What I Learned
This ticket demonstrated how startup programs impact system performance and how Task Manager can be used to identify resource bottlenecks.
